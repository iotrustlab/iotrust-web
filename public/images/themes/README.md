## Style Guidelines

- **Aspect ratio:** 16:9 (recommended)
- **Dimensions:** 1600×900 minimum, 1920×1080 preferred
- **File format:** JPG (optimized for web, ~200-500KB each)
- **Consistency:** Use similar visual language across all images
- **Text:** Avoid embedded text; images are supporting visuals
- **Lighting:** Maintain consistent contrast/brightness levels
- **Overlays:** Subtle gradients or technical overlays are OK

## Temporary Placeholders

Until final images are available, placeholder images should:
- Use subtle gradients matching the theme color scheme
- Include minimal abstract line art or geometric patterns
- Maintain the 16:9 aspect ratio
- Be visually distinct from each other
