Privacy-Preserving Edge Computing for Smart Cities

Funding Agency:

DOE

Amount:

Duration:

Project Overview

Investigating privacy-preserving computation techniques for edge devices in smart city deployments. Focus on homomorphic encryption and secure multi-party computation for sensor data processing.

Objectives

This project aims to develop novel privacy-preserving techniques that enable real-time data processing at the edge while maintaining strict privacy guarantees. We focus on homomorphic encryption schemes optimized for resource-constrained edge devices and secure multi-party computation protocols for collaborative data analysis.

Methodology

Our approach combines theoretical cryptographic research with practical system implementation. We develop lightweight homomorphic encryption libraries, design secure aggregation protocols, and evaluate performance on real smart city testbeds.
